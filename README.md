Strelka  — MVP Messenger v1.1

Минималистичный веб-мессенджер с фокусом на скорости общения.

## Что внутри
- Регистрация пользователя (frontend demo)
- Основной экран в стиле “записной книжки”
- Контакты и переключение диалогов
- Отправка сообщений (Enter)
- Демо-статусы: отправлено / доставлено
- Поиск по сообщениям активного диалога
- Демо-бот генерации ответов

## Запуск
Открыть `index.html` в браузере (или через Live Server).

## Ценность MVP
- Фокус на core-функции: быстрая связь
- Низкая когнитивная нагрузка
- Быстрый цикл тестирования гипотез с пользователями
