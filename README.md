Стрелка — MVP мессенджера быстрой связи

## Elevator Pitch
**Стрелочный** — это мессенджер с одной главной ценностью:  
**мгновенная и надежная доставка сообщений без перегруженного интерфейса.**

Мы не конкурируем в «развлекательных» функциях.  
Мы строим инфраструктуру быстрой коммуникации для людей и команд, где важна скорость принятия решений.

---

## Проблема
Современные мессенджеры перегружены вторичными механиками (контент, сторис, медиа-форматы), которые:
- отвлекают от основной задачи связи,
- увеличивают время до целевого действия (отправить/получить сообщение),
- усложняют UX для пользователей, которым нужна функциональность, а не «шум».

## Решение
Лаконичный мессенджер с фокусом на:
1. Быструю отправку текстовых сообщений,
2. Чистый интерфейс,
3. Понятный пользовательский сценарий.

---

## Статус продукта
- ✅ Frontend MVP landing page
- ⏳ Следующий этап: усложнение UI + состояния чатов
- ⏳ Затем: backend (auth, chat delivery, storage)

---

## Product Principles
- **Speed First**
- **Clarity over complexity**
- **One core use-case done right**

---

## Go-To-Market (черновик)
1. B2B пилоты с командами поддержки/операций.
2. Ниши с высокой стоимостью задержки ответа.
3. Тарифы: freemium + командные планы.

---

## Юридический блок
На лендинге размещен черновой блок оферты/правовой информации.  
Перед публичным запуском необходима адаптация под законодательство `РФ` (2026) с профильным юристом.

---

## Проверка названия
Требуется проверка:
- регистра компаний (юрлиц/ИП),
- реестры товарных знаков,
- доменные зоны,
- app stores / social handles.

> Финальное решение о доступности бренда принимается только после официальной проверки в реестрах `РФ`.
